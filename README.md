# Not Your Guru — Blog (Otomatik Yayın Sistemi)

Bu site tamamen ücretsiz GitHub Pages üzerinde çalışır ve **Salı & Cuma günleri
otomatik olarak** `queue/` klasöründeki bir sonraki yazıyı yayına alır.
Hiçbir şeye dokunmana gerek yok — sistem kendi kendine çalışır.

## Yeni bir yazı eklemek istersen (opsiyonel, ileride)

1. Yeni bir `.html` dosyası hazırla (Claude'dan iste, sana hazır verecektir).
2. Dosya adını sıradaki numarayla başlat: `003-yeni-yazi.html`, `004-...` gibi.
   (Sayı sırası = yayın sırası.)
3. GitHub'da bu repoya gir → `queue` klasörüne gir → **"Add file" → "Upload files"**
   → dosyayı sürükle bırak → **"Commit changes"**.
4. Bu kadar. Bir sonraki Salı ya da Cuma otomatik yayınlanır.

## Elle hemen yayınlamak istersen (beklemeden)

GitHub'da bu repo içinde üstteki **"Actions"** sekmesine gir →
**"Auto-publish next blog post"** iş akışını seç → sağ üstten
**"Run workflow"** butonuna bas. 1-2 dakika içinde yeni yazı yayında olur.
